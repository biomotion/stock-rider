# 🏍️ 台股騎士 StockRider TW

把台股 K 線圖當成越野賽道來騎的瀏覽器小遊戲（靈感來自 [StonkRider](https://stonkrider.com)）。
輸入股票代碼、選擇區間，股價走勢就會變成賽道：紅 K 是上坡、綠 K 是下坡，
波動越大路越難騎。騎多遠，「投資組合」就跟著那段行情漲跌。

## 執行

純靜態網頁，不需要安裝任何東西，起一個靜態伺服器即可：

```bash
python3 -m http.server 8741
# 開啟 http://localhost:8741
```

（必須走 http:// 而非 file://，因為使用了 ES modules。）

## 玩法

| 按鍵 | 功能 |
|---|---|
| `↑` / `W` | 油門 |
| `↓` / `S` | 煞車 |
| `←` / `A` | 後仰（翹孤輪；空中＝後空翻） |
| `→` / `D` | 前傾（點頭；空中＝前空翻） |
| `空白鍵` | 跳躍 |
| `N` / `Shift` | 氮氣加速 |
| `R` | 墜車後重來 |

頭部著地、或落地姿勢太歪就墜車。騎到終點旗即完賽。手機上會顯示觸控按鍵。

**特技計分**：空中翻轉落地得分，每圈 `500 × 連段倍率`，後/前空翻、Double/Triple…；
久滯空有「大跳躍」獎勵。連段倍率每成功一次 +1（最高 ×5），**墜車歸 1 並扣 500 分**。

**氮氣**：燃料約 2.4 秒，用完有冷卻、放開會回充，沿車身方向強力推進——配合上坡彈射可拉長滯空做連翻。

在選單或結算畫面閒置 10 秒會進入**自動演示模式**：AI 騎士（`js/autopilot.js`，
PD 平衡控制＋前方坡度預判減速）會隨機選股輪播騎乘，摔車自動重騎、完賽自動換股，
點擊或按任意鍵即可接手。

## 架構

- `index.html` / `style.css` — UI（選單、HUD、結算畫面、觸控鍵）
- `js/data.js` — 行情資料來源：[FinMind](https://finmindtrade.com) 公開 API
  （`TaiwanStockPrice` 日 K、`TaiwanStockInfo` 股名），免 token、支援 CORS，
  但未登入有流量限制，打太兇會暫時被擋。
- `js/terrain.js` — 收盤價 → 地形。重點：
  - 取對數價格再正規化，長區間的漲跌幅比例才正確
  - 依波動度自動決定地形總高度（第 90 百分位單日落差 ≈ 40° 坡）
  - FinMind 是未還原股價，除權息／分割會產生超過台股 ±10% 漲跌幅限制的
    人工缺口，這裡會把單日對數報酬夾限在 ±10%，避免出現垂直牆
  - Catmull-Rom 平滑取樣成高度場
- `js/game.js` — 自寫 2D 剛體物理（無外部套件）＋ Canvas 繪製、特技計分、氮氣。
  機車是單一剛體，前後輪用「彈簧＋阻尼」貼地，後輪驅動，每影格跑 10 個物理子步。
  操控手感參考 [StonkRider](https://stonkrider.com)（它用 Matter.js）的調校移植了三招防翻車：
  **角速度硬上限**（≈5.8 rad/s，車身轉不快所以翻轉永遠可控）、
  **雙輪著地時持續往地形坡度自動回正**、**空中強角速度阻尼**。
  落地時用累積旋轉角推算圈數計分。
- `js/main.js` — 流程串接、輸入處理、HUD 更新、自動演示。
