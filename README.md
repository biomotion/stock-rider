# 🏍️ 台股騎士 StockRider TW

把台股 K 線圖當成越野賽道來騎的瀏覽器小遊戲（靈感來自 [StonkRider](https://stonkrider.com)）。
輸入股票代碼、選擇區間，股價走勢就會變成賽道：紅 K 是上坡、綠 K 是下坡，
波動越大路越難騎。騎多遠，「投資組合」就跟著那段行情漲跌。

## 執行

純靜態網頁，不需要安裝任何東西，起一個靜態伺服器即可：

```bash
python3 -m http.server 8741
# 開啟 http://localhost:8741
```

（必須走 http:// 而非 file://，因為使用了 ES modules。）

## 玩法

| 按鍵 | 功能 |
|---|---|
| `↑` / `W` | 油門 |
| `↓` / `S` | 煞車 |
| `←` `→` / `A` `D` | 車身平衡（空中可前後翻轉） |
| `空白鍵` | 跳躍 |
| `R` | 墜車後重來 |

頭部著地就墜車。騎到終點旗即完賽。手機上會顯示觸控按鍵。

## 架構

- `index.html` / `style.css` — UI（選單、HUD、結算畫面、觸控鍵）
- `js/data.js` — 行情資料來源：[FinMind](https://finmindtrade.com) 公開 API
  （`TaiwanStockPrice` 日 K、`TaiwanStockInfo` 股名），免 token、支援 CORS，
  但未登入有流量限制，打太兇會暫時被擋。
- `js/terrain.js` — 收盤價 → 地形。重點：
  - 取對數價格再正規化，長區間的漲跌幅比例才正確
  - 依波動度自動決定地形總高度（第 90 百分位單日落差 ≈ 40° 坡）
  - FinMind 是未還原股價，除權息／分割會產生超過台股 ±10% 漲跌幅限制的
    人工缺口，這裡會把單日對數報酬夾限在 ±10%，避免出現垂直牆
  - Catmull-Rom 平滑取樣成高度場
- `js/game.js` — 自寫 2D 剛體物理（無外部套件）＋ Canvas 繪製。
  機車是單一剛體，前後輪用「彈簧＋阻尼」貼地，後輪驅動；
  每影格跑 10 個物理子步以維持穩定。空中沒按方向鍵時會緩緩轉向落點坡度（輔助）。
- `js/main.js` — 流程串接與輸入處理。
